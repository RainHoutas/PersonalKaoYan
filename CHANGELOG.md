# 更新日志 (Changelog)

## [Unreleased] / 最新更新

### 🎉 新特性 (Features)
- **TopNavbar 交互升级**: 新增全局顶栏导航（Top Navbar），内置毛玻璃背景（Glassmorphism）、平滑跟随遮罩效果（Segmented Control）以及支持点击拉开的详情浮层。
- **配置与进度自动对齐**: 在控制台页面的配置版面中支持配置“单科开始时间(startDate)”和“截止时间(endDate)”。增加直接覆盖修改“已完成模块”的能力，系统将自动补齐对应增减量的进度校准流水。
- **大盘进度条重构**: 为首页控制台（Dashboard）中的每一项科目进度条扩展了厚度，增加了进度数字的浮空映射（如 `20/50 进度`），使用混合模式字体颜色以最高对比度提升了数据展示性。

### 🔄 重构与优化 (Refactoring & UX)
- **释放视图宽度**: 全面废除原本占据左侧视线焦点的宽边栏 `<aside>` 以及阻挡视线的移动端底部导航。全应用转为竖向流式分布，为主体视图区域争取到了整整 `100%` 的渲染宽幅。
- **自适应修复**: 修订日历模式页面的容器栅格结构，在桌面宽屏环境中再度回归舒适的双栏（Left-Right）分布；并在小尺寸终端中安全降级为纵向排列。
- **项目目录扁平化**: 精简了过深的工程嵌套目录结构，将存放在 `kaoyan-tracker` 文件夹下侧的所有前端与后端相关源代码、包管理表向根目录提权平铺。全面拥抱最直接的单一存储库结构。

### 🛠 工程基建 (Chore)
- **代码库迁移**: 执行了项目级别的底层移动，整合了所有的 `.gitignore`。
- **一键托管部署**: 使用 Git 实现了对新工作区基线的快照切分和全量上传。
